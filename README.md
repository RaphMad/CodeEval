## Introduction

This project contains solutions and tests for the coding challenges provided by www.codeeval.com.
The project is based on www.haskellstack.org.

## Build

Execute `stack build`.

## Run

Adapt **app\Main.hs** to point to the desired example, then execute `stack exec CodeEval.exe .\input\<inputfile>`.

## Test

Execute `stack test`.
